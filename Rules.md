
# 탁송 수신지 자동 생성 페이지

## 페이지 정의
1. 광역시/도를 입력한다 (버튼 혹은 Select box) 
2. 해당 광역시/도 하위 행정구역 (시,군,구)를 입력한다.
   -(https://www.data.go.kr/data/15059464/openapi.do) 오픈API로 행정구역도(광역시도, 시군구, 읍면동, 리)의 정보를 공간정보
3. 생성버튼을 통해 해당 행정구역내 실제 아파트, 빌라의 주소를 갯수만큼 생성한다.
4. 이때 https://postcode.map.daum.net/guide 를 참고하여 사용한다 
5. oncomplete 속성의 반환데이터의 apartment 필드(Y/N	공동주택 여부 (아파트,연립주택,다세대주택 등))를 이용하여 필터링 하는방법으로 사용한다
6. 생성된 주소를 복사할 수 있는 기능을 탑재한다. 

## 규칙
1. 생성된 코드에는 한글로 주석을 작성합니다.
2. 추후 유지보수에 용이 하게 작성합니다.
3. 결과물에 필요한 정보 ( api key, .env 파일내 환경 변수) 등이 있다면 사용자가 어디에 어떤정보를 입력해야 하는지 보고합니다.
4. 환경변수 사용시 운영서버와 개발서버용 환경변수를 구분합니다.
