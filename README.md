## 주요 기능

- 출발지 기본값 제공 및 재설정 버튼
- 다음 우편번호 서비스 연동
- 공동주택(apartment=Y) 주소만 수집
- 최근 선택한 주소 최대 10개 저장 및 원클릭 복사
=======
- 브라우저가 자동으로 열리며, `http://localhost:5173`에서 확인할 수 있습니다.
- 출발지를 입력하고 검색하면 서버의 `/api/road-address`를 통해 도로명 주소 결과를 가져옵니다.
- 결과 목록에서는 빌라와 아파트가 필터링되어 표시됩니다.
